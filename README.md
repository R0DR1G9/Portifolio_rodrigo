# Portfólio — Rodrigo Barbosa

Site de portfólio pessoal em HTML5, CSS3 e JavaScript puro. Projeto 3 — Front-End, CESAR School (2026).

## Estrutura

```
portfolio/
├── index.html       # Página inicial
├── sobre.html       # Sobre mim, tecnologias, links
├── conquistas.html  # Timeline de conquistas com filtro
├── style.css        # Estilos (CSS variables, Grid, Flexbox)
├── main.js          # JS: scroll reveal, menu mobile, filtro
└── README.md
```

## Critérios atendidos

- Layout com **Flexbox e CSS Grid** em todas as páginas
- **Tags semânticas** — `<nav>`, `<main>`, `<header>`, `<section>`, `<article>`, `<footer>`, `<time>`, `<blockquote>`, `<ol>`
- **Navegação entre 3 páginas** — index, sobre, conquistas
- **JavaScript**: scroll reveal via IntersectionObserver, menu mobile toggle, filtro de timeline por categoria
- **README** com mais de um commit de atualização

## Como rodar

Abrir `index.html` em qualquer navegador moderno. Sem dependências, sem build.

## Adicionar foto

Coloque a imagem em `assets/rodrigo.jpg` — o site exibe automaticamente na página Sobre.
